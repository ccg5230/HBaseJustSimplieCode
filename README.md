Springboot使用`hbase-client`包连接HBase2.X 
====================================================================

# 项目版本
HBase2.2.4 + Hadoop2.8.5 +zookeeper3.6.0

# 项目运行
使用maven打包，注意关闭maven打包运行test，然后上传到hbase安装环境运行，Java -jar xxx.jar &
